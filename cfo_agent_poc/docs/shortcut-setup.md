# iPhone 快捷指令搭建指南

整个采集链路里，唯一需要你动手搭建的就是这个快捷指令。它只做两件事：**截图当前屏幕 → 把截图作为邮件附件发出去**。搭好之后，日常使用就是「支付 → 打开账单页 → 点一下」。

> 💡 不想手搭？[📲 一键安装现成的快捷指令](https://www.icloud.com/shortcuts/eb909451885a42208195397d54db34a0)，装好后只需把「发送电子邮件」的收件人改成你自己的同步邮箱（下文「二、搭建步骤」第 3 步），即可跳过手动搭建。

## 一、准备

- 一个开通了 IMAP 的邮箱（如 QQ 邮箱），与 `cfo_agent_poc/.env` 里 `CFO_MAIL_USER` 配置的是同一个
- iPhone 自带的「快捷指令」App（Shortcuts）

> 建议专门用一个邮箱收账单截图，避免和日常邮件混在一起。发件人可以是同一个邮箱（自己发给自己）。

## 二、搭建步骤

1. 打开「快捷指令」App → 右上角 **+** 新建快捷指令
2. 添加操作：搜索 **「截屏」**（Take Screenshot），加入
3. 添加操作：搜索 **「发送电子邮件」**（Send Email），加入，然后逐项配置：
   - **收件人**：你的同步邮箱（`.env` 里的 `CFO_MAIL_USER`）
   - **主题**：`CFO_CAPTURE_SCREENSHOT`（必须与 `.env` 的 `CFO_MAIL_SUBJECT` 完全一致）
   - **正文**：留空即可
   - **附件**：选择上一步的「截屏」变量
   - 关闭 **「运行时显示」**（Show Compose Sheet），这样点一下就直接发送、不弹窗
4. 给快捷指令命名（如「记一笔」），建议：
   - 添加到主屏幕，或
   - 设置「辅助功能 → 触控 → 轻点背面」双击触发，体验最顺滑

## 三、日常使用

1. 微信 / 支付宝支付完成后，**打开该笔账单的详情页**
2. 确认页面里能看到：**金额、支付时间、商户/商品、支付方式**（这些是解析必需字段）
3. 触发快捷指令 → 完成。剩下的交给 Mac 端：同步邮件 → 本地 OCR → 解析入账

> 小技巧：微信在「我 → 服务 → 钱包 → 账单」，支付宝在「我的 → 账单」里都能补录历史交易——逐笔打开详情页触发快捷指令即可回填旧账。

## 四、导出 iCloud 分享链接（项目维护者）

想让 README 里出现「一键安装」链接，需要把你的快捷指令通过 iCloud 分享出去：

1. 快捷指令 App → 长按你的快捷指令 → **分享**
2. 选择 **「拷贝 iCloud 链接」**（Copy iCloud Link）
3. 得到形如 `https://www.icloud.com/shortcuts/xxxxxxxx` 的链接，替换 README 里的占位链接

**分享前务必检查**：把快捷指令里的收件人邮箱改成占位符（如 `your-email@example.com`）再导出，或在 README 里注明安装后需要修改收件人——不要把你的私人邮箱地址随链接公开。

## 常见问题

| 问题 | 排查 |
|---|---|
| 同步后没有新交易 | 邮件主题是否与 `CFO_MAIL_SUBJECT` 完全一致；邮件是否未读（已处理的邮件会被标记已读） |
| 解析字段不全 / 置信度低 | 截图时账单详情页是否完整展示了金额、时间、商户；截图不要截半屏 |
| 想不走邮箱手动补账 | `python3 process_bill_image.py /path/to/bill.png --source manual` |
